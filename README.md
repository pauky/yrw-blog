# yrw-blog
yrw-blog服务端
express版本

## Installation

#### 安装依赖
安装mongodb

下载依赖包
```
$ npm install
```

#### 前端资源
下载项目[yrw-blog-static](https://github.com/pauky/yrw-blog)，并按照readme.md的指示编译前端资源，再按照项目在本地的地址，配置conf/config.js的staticRoot值。

#### 启动
```
$ node blog-portal-koa-server.js
```
访问[localhost:3100](http://localhost:3100)
